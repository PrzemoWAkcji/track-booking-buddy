================================================================================
SYSTEM REZERWACJI TOROW - WERSJA PRZENOŚNA
================================================================================

INSTRUKCJA UŻYTKOWANIA:

1. Rozpakuj folder System-Rezerwacji-Torow na dowolne miejsce na dysku

2. Uruchom aplikację:
   - Kliknij dwukrotnie na plik: run.bat
   
3. Przy pierwszym uruchomieniu:
   - System pobierze niezbędne zależności (~200MB)
   - Poczekaj aż skończy instalacji
   - Aplikacja otworzy się w oknie

4. Po uruchomieniu:
   - Aplikacja pracuje OFFLINE - nie potrzebuje internetu
   - Wszystkie dane są przechowywane lokalnie
   - Harmonogramy są archiwizowane w przeglądarce

WYMAGANIA:
- Windows 10 lub nowszy (64-bit)
- ~500MB wolnego miejsca na dysku (dla zależności)
- Internet przy pierwszym uruchomieniu

FUNKCJE:
✓ Zarządzanie rezerwacjami torów
✓ Generowanie PDF harmonogramów
✓ Archiwizowanie tygodniowych harmonogramów
✓ Export do CSV (Excel)
✓ Wersja RODO (bez nazw kontrahentów)
✓ Wersja portable - bez instalacji

ROZWIĄZYWANIE PROBLEMÓW:

Jeśli aplikacja się nie uruchamia:
1. Sprawdź czy masz zainstalowany Node.js
2. Kliknij dwukrotnie run.bat jeszcze raz
3. Sprawdź czy Windows Defender nie blokuje aplikacji
